---
title: FollowupSpread
---

Represents the spread behavior of a weapon, including how spread increases per shot, maximum spread values, and dissipation timing.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `spreadGain` | `float` | The spread gain per shot. |
| `maxSpreadHip` | `float` | Maximum spread when hip firing. |
| `maxSpreadAim` | `float` | Maximum spread when aiming. |
| `spreadStayTime` | `float` | Time for spread to stay at max. |
| `spreadDissipateTime` | `float` | Time for spread to dissipate. |
| `proneMultiplier` | `float` | Multiplier applied when prone. |
