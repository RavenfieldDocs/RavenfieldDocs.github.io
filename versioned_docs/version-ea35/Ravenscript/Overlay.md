### ShowMessage

Displays a text message as an overlay on the player's screen for a specified duration.

| Parameter | Type | Description |
|-----------|------|-------------|
| `text` | `string` | The text to display. |
| `duration` | `float` | How long to show the message in seconds (clamped to 3.5-30). |
