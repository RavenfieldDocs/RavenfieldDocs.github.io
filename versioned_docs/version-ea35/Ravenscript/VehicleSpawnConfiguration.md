Represents a configuration for spawning vehicles, specifying the rarity tier and vehicle type.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `tier` | `RarityTier` | The rarity tier for the vehicle spawn. |
| `type` | `VehicleSpawnType` | The type of vehicle to spawn. |
