Represents a Unity UI Image component in Ravenscript. Inherits from [Graphic](./Graphic.md).

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `enabled` | `bool` | Whether the Image component is enabled. |

## Methods

## Events

| Event | Signature | Description |
|-------|-----------|-------------|
| `onPointerEnter` | `ScriptEvent` | Fired when the pointer enters the image area. |
| `onPointerExit` | `ScriptEvent` | Fired when the pointer exits the image area. |
| `onPointerClick` | `ScriptEvent<int>` | Fired when the pointer clicks on the image. The callback receives the pointer button index. |
| `onPointerDown` | `ScriptEvent<int>` | Fired when the pointer is pressed down on the image. The callback receives the pointer button index. |
| `onPointerUp` | `ScriptEvent<int>` | Fired when the pointer is released on the image. The callback receives the pointer button index. |
