Represents a Unity UI RawImage component, used for displaying textures.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `enabled` | `bool` | Whether the RawImage is enabled. |

## Events

| Event | Signature | Description |
|-------|-----------|-------------|
| `onPointerEnter` | [ScriptEvent](./ScriptEvent.md) | Fired when the pointer enters the RawImage. |
| `onPointerExit` | [ScriptEvent](./ScriptEvent.md) | Fired when the pointer exits the RawImage. |
| `onPointerClick` | `ScriptEvent<int>` | Fired when the pointer clicks the RawImage. |
| `onPointerDown` | `ScriptEvent<int>` | Fired when the pointer presses down on the RawImage. |
| `onPointerUp` | `ScriptEvent<int>` | Fired when the pointer releases on the RawImage. |
