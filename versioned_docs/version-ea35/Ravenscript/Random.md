### RangeInteger

Generates a random integer in the given range.

| Parameter | Type | Description |
|-----------|------|-------------|
| `minInclusive` | `int` | The inclusive lower bound of the random number returned. |
| `maxExclusive` | `int` | The exclusive upper bound of the random number returned. |

[return: int]
A random integer greater than or equal to `minInclusive` and less than `maxExclusive`.
