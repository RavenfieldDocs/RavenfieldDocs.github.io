## Methods

### SetModContentFileURL

Sets the VideoPlayer url to a file in the mod content folder. This is the same as setting the VideoPlayer url to a local file path via the default Unity API. For example, `cool_video.mov` will load the file cool_video.mov next to your mod rfc file.

| Parameter | Type | Description |
|-----------|------|-------------|
| `localPath` | `string` | The relative path to the video file within the mod content folder. |
