---
title: Button
---

Represents a Unity UI Button component accessible from Ravenscript.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `onClick` | `ScriptEvent` | Fires when the button is clicked. |
| `onPointerEnter` | `ScriptEvent` | Fires when the pointer enters the button area. |
| `onPointerExit` | `ScriptEvent` | Fires when the pointer exits the button area. |
| `onPointerClick` | `ScriptEvent<int>` | Fires when the pointer clicks on the button. The callback receives the pointer ID. |
| `onPointerDown` | `ScriptEvent<int>` | Fires when the pointer is pressed down on the button. The callback receives the pointer ID. |
| `onPointerUp` | `ScriptEvent<int>` | Fires when the pointer is released on the button. The callback receives the pointer ID. |

## Events

| Event | Signature | Description |

| `onClick` | `ScriptEvent` | Triggered when the button is clicked. |

| `onPointerUp` | `(button)` | Triggered when the pointer is released on the button. The `button` parameter is the pointer ID. |
