### Cast

Cast the object to the target type. Returns the cast object, or `nil` on failure.

| Parameter | Type | Description |
|-----------|------|-------------|
| `obj` | `object` | The object to cast. |
| `type` | `object` | The target type to cast to. |

[return: object?]
The cast object, or `nil` on failure.
