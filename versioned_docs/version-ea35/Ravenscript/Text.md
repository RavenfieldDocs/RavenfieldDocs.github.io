---
title: Text
---

Represents a Unity UI Text component accessible from Ravenscript.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `onPointerEnter` | `ScriptEvent` | Fires when the pointer enters the text element area. |
| `onPointerExit` | `ScriptEvent` | Fires when the pointer exits the text element area. |
| `onPointerClick` | `ScriptEvent<int>` | Fires when the pointer clicks on the text element. The callback receives the pointer button index (0=left, 1=right, 2=middle). |
| `onPointerDown` | `ScriptEvent<int>` | Fires when the pointer is pressed down on the text element. The callback receives the pointer button index. |
| `onPointerUp` | `ScriptEvent<int>` | Fires when the pointer is released on the text element. The callback receives the pointer button index. |
