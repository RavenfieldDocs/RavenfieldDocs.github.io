title: Ladder

Represents a ladder that actors can climb.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `height` | `float` | The height of the ladder. |
| `topPosition` | `Vector3` | Returns the top position of the ladder in world space. |
| `bottomPosition` | `Vector3` | Returns the bottom position of the ladder in world space. |
| `topExitPosition` | `Vector3` | Returns the top exit position of the ladder in world space. |
| `bottomExitPosition` | `Vector3` | Returns the bottom exit position of the ladder in world space. |
