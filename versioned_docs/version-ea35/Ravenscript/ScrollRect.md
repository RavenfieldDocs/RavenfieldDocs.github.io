---
title: ScrollRect
---

Wrapper for Unity's `ScrollRect` UI component, providing pointer event access.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `onPointerEnter` | [ScriptEvent](./ScriptEvent.md) | Fired when the pointer enters the scroll rect area. |
| `onPointerExit` | [ScriptEvent](./ScriptEvent.md) | Fired when the pointer exits the scroll rect area. |
| `onPointerClick` | `ScriptEvent<int>` | Fired when a pointer click occurs on the scroll rect. |
| `onPointerDown` | `ScriptEvent<int>` | Fired when a pointer is pressed down on the scroll rect. |
| `onPointerUp` | `ScriptEvent<int>` | Fired when a pointer is released on the scroll rect. |

## Events

| Event | Signature | Description |
|-------|-----------|-------------|
| `onPointerEnter` | [ScriptEvent](./ScriptEvent.md) | Fired when the pointer enters the scroll rect area. |
| `onPointerExit` | [ScriptEvent](./ScriptEvent.md) | Fired when the pointer exits the scroll rect area. |
| `onPointerClick` | `(button)` | Fired when a pointer click occurs. The `button` parameter indicates which mouse button was clicked. |
| `onPointerDown` | `(button)` | Fired when a pointer is pressed down. The `button` parameter indicates which mouse button was pressed. |
| `onPointerUp` | `(button)` | Fired when a pointer is released. The `button` parameter indicates which mouse button was released. |
