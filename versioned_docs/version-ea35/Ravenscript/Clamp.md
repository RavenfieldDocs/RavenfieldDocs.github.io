---
title: Clamp
---

Defines pitch or yaw angular constraints for a mounted stabilized turret.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `enabled` | `bool` | Whether this clamp constraint is active. |
| `max` | `float` | The maximum angle in degrees for this constraint. |
| `min` | `float` | The minimum angle in degrees for this constraint. |
