---
title: MountedStabilizedTurret
---

Represents a stabilized turret that inherits from [MountedWeapon](./MountedWeapon.md), with spring-based recoil, rotation clamping, and stabilization features.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `aimCamera` | [Camera](https://docs.unity3d.com/ScriptReference/Camera.html) | The aim camera for the mounted weapon. |
| `overrideCamera` | [Camera](https://docs.unity3d.com/ScriptReference/Camera.html) | The override camera for the mounted weapon. |
| `seat` | [Seat](./Seat.md) | The seat associated with the mounted weapon. |
