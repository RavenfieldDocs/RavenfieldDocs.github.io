Use these values to Enable/Disable built in game features.

## Properties

| Property | Type | Description |
|----------|------|-------------|
| `actorHitEffect` | `bool` | Controls actor damage particle effect emission. |
| `actorBloodDrops` | `bool` | Controls actors dropping blood drop particles on damage. |
| `impactDecals` | `bool` | Visibility of impact decals. |
| `bloodDecals` | `bool` | Visibility of blood decals. |
